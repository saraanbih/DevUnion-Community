# 🌐 DevUnion Community

**DevUnion** is a student-led tech community created to **support each other, learn together, and grow collectively**.  
Whether you're exploring frontend, backend, UI/UX, cybersecurity, mobile development, or any tech-field, this community will help you stay on track and build real-world skills alongside your peers.

We believe in **collaborative learning**, open discussions, and an environment where everyone contributes and benefits.

## 📌 Community Rules

### 1. 🤝 Mutual Respect
- Speak politely and professionally with others.
- Avoid using inappropriate or disrespectful language.
- Listen to each other during discussions and avoid interrupting.

### 2. 📚 Channel Discipline
- Use each channel only for its intended purpose.
- Read the pinned messages and follow the structure.

### 3. 🤲 Support Each Other
- Share useful information, resources, and guidance.
- Encourage and help others, especially newcomers.
- No discouraging replies or toxic behavior.

### 4. 🚫 No Spam or Irrelevant Content
- No advertising or repeated messages.
- Avoid off-topic discussions in technical channels.

### 5. 🎙 Voice Chat Etiquette
- Keep your mic muted when not speaking.
- Let others talk and avoid shouting or interrupting.
- Stay on topic during voice discussions.

### 6. 🛡 Follow Moderator Instructions
- Moderators are here to guide and support. If you disagree with something, kindly request clarification or discuss it respectfully.
- Ignoring rules may result in warnings or removal from the server.

## 🧾 Server Roles & Responsibilities

- **Member** - Default role for all students in the community.
- **Frontend Moderator** - Guides frontend members, offers roadmaps, and supports learning paths.
- **UI-UX Moderator** - Helps learners in design, provides UX/UI resources and direction.
- **.Net Moderator** - Supports backend development with .NET and offers learning tracks.
- **Django Moderator** - Provides guidance in Django/Python and shares practical roadmaps.
- **Node Js Moderator** - Helps backend learners with Node.js and offers structured support.
- **Cyber Security Moderator** - Offers mentorship in security tracks like SOC and pentesting.
- **Flutter Moderator** - Assists mobile devs with Flutter and provides learning paths.
- **Trusted** - Recognized helpful members who actively support the community.
- **Frontend** - Members learning or working in frontend development.
- **.Net** - Backend developers or learners using the .NET stack.
- **Django** - Students focusing on Django and Python backend.
- **Node Js** - JavaScript backend learners and developers.
- **Flutter** - Mobile developers using the Flutter framework.
- **UI-UX** - Designers and researchers focused on user experience and interface design.
- **Cyber Security** - Members interested in ethical hacking, SOC, and security tools.
- **@everyone** - Default role given to all server members.
